Project Overview

This project collects publicly available match data from the Riot Games API across all competitive League of Legends regions. All data is aggregated, anonymized, and used exclusively for educational and research purposes. No raw data is commercially distributed or resold. The project fully complies with the Riot Games API Terms of Service.

Research Areas

  • Meta analysis across game patches
  • Champion win rate and scaling curves
  • Team composition synergies
  • Regional playstyle differences
  • Gold and experience power curves
  • Match outcome prediction models

Technical Implementation

  • Token-bucket rate limiter with exponential backoff for API compliance
  • Multi-threaded architecture with Redis-based state management
  • Incremental Parquet writes with fixed schemas for data consistency
  • LRU caching to deduplicate processed matches
  • Graceful shutdown with checkpoint persistence
